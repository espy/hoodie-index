This is Hoodie:

Hoodie is a free, Open Source App Server that abstracts away backend tasks which are usually required when building applications, no matter what the final use case for the app will be. It gives you a compact API which has built-in user signup and administration, data storage, loading, shares, synchronisation and sending emails. With Hoodie, all you do when building an app is develop the frontend code, hook it up to Hoodie’s frontend API, and you’re ready. Hoodie can be used for building applications for the web and iOS.
